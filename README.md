# alef-rails

alef-rails provides the [Alef](http://alef.hagilda.com/) web font as a Rails
engine for use with the asset pipeline.

## Installation

Add this line to your application's Gemfile:

    gem 'alef-rails'

And then execute:

    $ bundle

## Usage

In your `application.css`, include the css file:

```css
/*
 *= require alef
 */
```

or use sass:

```scss
@import 'alef';
```
